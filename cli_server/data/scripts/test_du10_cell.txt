상위 : lncel
하위 : drx, srdx, sib
****************공식****************
lcrId : 0~255
nbIotLinkedCellId : macro는 nbiot의 lcrId, nbiot는 macro의 lcrId
tac : getScf파일 읽어서 하는것이고 어떤것을 참조할지는 더 봐야함
prodCodePlanned : RU type에 따라서 다름
phyCellId : macro와 nbiot 둘 서로 값이 같아야함
CABLINK : genScf중 없는 id를 할당
firstEndpointDN : genScf에서 다른
