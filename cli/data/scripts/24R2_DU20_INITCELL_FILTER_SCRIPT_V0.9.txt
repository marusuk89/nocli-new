##!
## EQM-1/APEQM-1/ANTL-?  은  CLI 엔진이 User Input CLI를 읽어서... 각 RU 별 Antenna Path 특성에 따라 프로그램에서 자동으로 구성한다.
## 이때 Init-cell 모드에서 user가 입력한 ru-type에 따른다.
## MNL-1/MNLENT-1/CELLMAPPING-1/LCELL-?도  ANTL과 마찬가지로 CLI 입력값(User CLI Input)에 맞춰서 프로그램이 자동으로 구성한다.

EQM  1
  APEQM  1
    #RMOD  000
      #auto-config  ANTL  *
    exit-all


MNL  1
  MNLENT 1
    CELLMAPPING  1
      #auto-config  LCELL  *
    exit-all
